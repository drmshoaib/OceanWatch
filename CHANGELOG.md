### Added

- GitHub Actions CI workflow for Windows and Ubuntu CMake builds.

### Existing Project Capabilities

- Robust AIS CSV loading with malformed-row handling.
- Vessel track grouping and timestamp ordering.
- Geospatial utilities for trajectory analysis.
- Feature engineering for speed, turning, distance, duration, and AIS gaps.
- Rule-based risk scoring with explanations.
- Lightweight statistical anomaly detection.
- Circular protected-area proximity analysis.
- CSV and Markdown report generation.
- Catch2 unit tests integrated with CTest.
